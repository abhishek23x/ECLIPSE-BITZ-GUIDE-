* NOW Install Bitz CLI
```bash
cargo install bitz
```

```bash
screen -S eclipse
```

```bash
bitz collect
```
Send 0.005 $ETH on Eclipse to start mining.

Other Commands:

  •	Claim your Bitz:
```bitz claim```
  
  •	Check your balance:
```bitz account```

  • Config CPU (change the number to change cores):
```bitz collect --cores 8```
  
  •	View all commands:
```bitz -h```
